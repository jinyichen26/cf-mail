# 商品购买网站规格说明

## 一、项目概述

### 项目名称
商品购买网站

### 核心功能
一个极简的商品购买界面，展示三个商品，点击购买后显示支付宝收款码和订单信息。

## 二、功能需求

### 2.1 页面布局
- 左上角显示应用名称
- 页面中央展示三个商品
- 极简设计，无多余元素

### 2.2 商品展示
- 三个商品卡片并排显示
- 每个商品包含：图片、名称、价格
- 每个商品有"购买"按钮

### 2.3 购买功能
- 点击购买按钮后，弹出购买面板
- 面板左侧：支付宝收款码
- 面板右侧：
  - 联系邮箱
  - 支付订单号
  - 商品信息摘要

## 三、技术方案

### 前端技术
- HTML5 + CSS3 + JavaScript（单文件）
- 极简CSS样式

### 数据管理
- 商品数据使用硬编码JSON
- 订单号随机生成
- 无需持久化存储

## 四、验收标准

### 功能验收
- [ ] 左上角显示应用名称
- [ ] 页面中央显示三个商品
- [ ] 商品信息完整（图片、名称、价格）
- [ ] 购买按钮可点击
- [ ] 点击购买后弹出支付面板
- [ ] 面板左侧显示支付宝收款码
- [ ] 面板右侧显示联系邮箱
- [ ] 面板右侧显示支付订单号
- [ ] 订单号为随机生成

### 界面验收
- [ ] 布局简洁美观
- [ ] 商品排列整齐
- [ ] 支付面板左右布局清晰
